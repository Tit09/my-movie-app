Ce projet est une mini-application web permettant d'explorer les films en utilisant l'API TMDB. 
Réalisé dans le cadre d'un apprentissage sur les outils d'IA et le développement frontend.

Fonctionnalités
Affichage des films populaires, mieux notés et à venir.
Barre de recherche interactive.
Interface moderne en mode sombre.
Cartes de films avec notes et dates de sortie.
Outils utilisés
IA : ChatGPT pour la structure du code et la documentation.
Code : VS Code, HTML, CSS, JavaScript.
Hébergement : GitHub Pages.
